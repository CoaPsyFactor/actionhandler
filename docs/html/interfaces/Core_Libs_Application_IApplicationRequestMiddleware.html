Core\Libs\Application\IApplicationRequestMiddleware

Implementing this interface into "IApplicationRequestHandler" will tell "Application" that it should execute middlewares added to "Middleware" class before request handler is executed. If there are some unfinished (not executed) middlewares, script will prevent request handler to be executed, and finish request (send response)
